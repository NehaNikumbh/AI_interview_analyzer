# AI_interview_analyzer
AI Interview Analyzer is a web-based application that helps students and job seekers practice interviews and improve their communication skills. The system asks interview questions, records the user's responses through a webcam and microphone, and analyzes various aspects of the interview performance.

The application uses Artificial Intelligence and Computer Vision techniques to evaluate facial expressions, eye contact, posture, confidence level, speaking speed, and response quality. Based on the analysis, it generates a detailed feedback report highlighting strengths and areas for improvement.
